# schjonhaug.dev

Source of the Jekyll blog at <schjonhaug.dev>.

## Initial setup

### Prerequisites

- [Bundler](https://bundler.io)

### Setup

```shell
bundle install
```

### Run locally

```shell
bundle exec jekyll serve
```

### Check for updates

```shell
bundle outdated
```

and then, in case of updates

```shell
bundle update
```
